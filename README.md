# Blanja Web Application

Blanja is an e-commerce web application that sells many items and users can select the items they want and then proceed to payment. There is also a seller account for adding and changing products.

<br />

## Features

- Users can create a store account and a customer account.
- Users can edit their profile (store or customer).
- Users can browse available products.
- Store accounts can create product listings for items they are selling.
- Store accounts can edit and delete products.

## Built With

- React JS
- Bootstrap
- Redux Toolkit

## Installation

Clone the project

```bash
  git clone https://github.com/mrizkysolehudin/fe-blanja-TeamProject my-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```
